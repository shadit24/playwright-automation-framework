# Playwright Automation Framework

## Overview
This project is an end-to-end test automation framework built with Playwright.

The goal of this framework is to demonstrate UI and API automation testing practices.

## Technologies

- Playwright
- JavaScript
- Node.js
- Git
- REST API Testing

## Project Structure
playwright-automation-framework
│
├── tests
│ └── Test scenarios
│
├── pages
│ └── Page Object Model classes
│
├── utils
│ └── Helper functions
│
└── data
└── Test data

## Testing Approach
